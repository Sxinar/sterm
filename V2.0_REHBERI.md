### 🔧 Komut Kategorileri

#### 1️⃣ SİSTEM & GÜNCELLEME
```
guncelle          - GitHub'dan otomatik güncelle
surum             - Versiyon detaylarını göster
analiz            - Derin sistem raporu
bilgisayar        - Bilgisayar bilgileri
saat              - Tarih ve saat
temizle           - Ekranı temizle
```

#### 2️⃣ DOSYA & DİZİN YÖNETİMİ
```
yd [dizin]        - Dizin değiştir (cd)
yarat [isim]      - Yeni klasör oluştur
sil [isim]        - Dosya/klasör sil
gezgin            - Dosyaları renkli listele
ara [isim]        - Dosya/klasör ara
boyut [isim]      - Dosya boyutunu KB cinsinden göster
oku [dosya]       - Dosya içeriğini oku
kopyala [kaynak]  - Dosya kopyala (cp -r)
tası [hedef]      - Dosya taşı (mv)
```

#### 3️⃣ AKILLI ARAÇLAR
```
hava [sehir]      - Hava durumunu göster (wttr.in API)
hesapla [işlem]   - Matematiksel işlem çöz (2+2*5 gibi)
parola-uret [n]   - n karakterlik güvenli şifre (default: 12)
indir [url] [ad]  - İnternetten dosya indir (requests gerekli)
web [url]         - Tarayıcıda site aç
ip_bul            - Yerel IP adresini göster
ping [host]       - Bağlantı testi yap
disk              - Disk kullanımı göster
not-al [metin]    - Hızlı not kaydet
notları-gör       - Tüm notları görüntüle
```

#### 4️⃣ PAKET & YÖNETİM
```
serix-get [paket] - Paket yükleyici (Apt/Pacman/Brew/Winget)
serix-kök         - Yönetici/Root erişimi
kapat             - Bilgisayarı kapat
yeniden           - Yeniden başlat
kilitle           - Ekranı kilitle
```

#### 5️⃣ GEÇMİŞ & AYARLAR
```
gecmis [n]        - Son n komutun geçmişini göster
ayar [anahtar]=   - Ayarları yönet ve kaydet
config            - Mevcut konfigürasyonu göster
```

#### 6️⃣ ÇIKIŞ
```
yardim / yardım   - Tam komut rehberini göster
çıkış             - STerm'den ayrıl
```

## 🎨 KOMUT ÖRNEKLERİ

### Dosya Yönetimi
```bash
yd /home/user          # Home'a git
yarat yeni_klasor      # Yeni klasör oluştur
gezgin                 # Dosyaları listele
ara config             # config içeren dosyaları ara
boyut document.pdf     # PDF boyutunu gör
oku README.md          # Dosya oku
```

### Araçlar
```bash
hava Istanbul          # İstanbul'un hava durumu
hesapla 100*5/2        # 250 sonucu ver
parola-uret 16         # 16 karakterlik şifre oluştur
web google.com         # Google'u tarayıcıda aç
ip_bul                 # IP adresini göster
disk                   # Disk bilgisi
```

### Sistem
```bash
analiz                 # Sistem raporu
saat                   # Güncel zaman
guncelleme             # Güncellemeleri kontrol et
serix-get python3      # Python3 kur
```

---

## ⚙️ KONFİGÜRASYON

Ayarlar `sterm_config.json` dosyasında JSON formatında saklanır:

```json
{
  "tema": "dark",
  "dil": "tr",
  "hava_sehri": "Konya",
  "otomatik_guncelle": false,
  "sesin_aç": true,
  "tarih_formatı": "%d/%m/%Y %H:%M:%S"
}
```

### Ayarı Değiştirme
```bash
ayar hava_sehri=Ankara      # Hava şehrini değiştir
ayar tema=light             # Temayı değiştir
ayar otomatik_guncelle=true # Otomatik güncellemeyi aç
config                      # Tüm ayarları göster
```
