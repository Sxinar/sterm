## ❓ SSS (Sık Sorulan Sorular)

### S: Komutlar büyük/küçük harfe duyarlı mı?
**C**: Hayır, `Yardim`, `YARDIM`, `yardim` hepsi aynı işlem yapar.

### S: Argüman hatasına ne yapmalı?
**C**: Komutun yanında `⚠️ Kullanım:` yazarsa okunur:
```bash
# Hata verirse:
⚠️ Kullanım: indir [url] [dosya_adı]

# Doğru şekil:
indir https://example.com/file.zip myfile.zip
```

### S: Ayarlar nerede saklanır?
**C**: `sterm_config.json` dosyasında, programın aynı klasöründe.

### S: Notlarım nereye kaydediliyor?
**C**: `sterm_notlar.txt` dosyasında.

### S: Komut geçmişini temizleyebilir miyim?
**C**: Evet, `sterm_history.txt` dosyasını silebilirsin.

### S: "Permission denied" hatası alıyorum
**C**: Dosyayı çalıştırılabilir yap:
```bash
chmod +x sterm_v2.0.py
```

### S: Windows'ta çalışmıyor
**C**: Python'u kur ve şu komutla çalıştır:
```bash
python sterm_v2.0.py
```
